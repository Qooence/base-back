## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
npm run dev
```
### Production(Build)
```bush
npm run build
```

## 简介
&emsp;&emsp;智慧家居后台管理平台前端页面是基于iview Admin的一套后台集成解决方案，风格统一，设计考究。实现了动态增加路由，更贴合前后端分离开发的实际需求。

## 功能

- 登录/登出
- 设备管理
    - 设备管理
    - 遥控器管理
    - 配件管理
    - 防区管理
    - 报警记录
- 系统管理
    - 用户管理
    - 角色管理
    - 菜单管理
    - 字典管理
    - 系统日志
- 多级路由展示
    - 二级一
    - 二级二
      - 三级一
      - 三级二
    - 二级三
- 错误页面
    - 403页面
    - 404页面
    - 500页面
- 高级路由
    - 动态路由
    - 带参页面
- 收缩侧边栏
- tag标签导航
- 面包屑导航
- 全屏/退出全屏
